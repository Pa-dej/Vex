# Changelog

## 0.1.1 - 2026-03-30

- Fixed repository/readme links to the real GitHub project URL (`pa-dej/Vex`).

## 0.1.0 - 2026-03-30

- Initial standalone `vex-proxy-sdk` extraction from the Vex proxy repository.
- Stable plugin trait (`VexPlugin`) and event bus API.
- Full proxy event set (connection, backend, status, plugin messaging, reload, health).
- Player transfer API (`ProxiedPlayer::transfer`) and per-player metadata helpers.
- Command registry and optional Prometheus metrics registration API.
- Prelude module and crate-level plugin quickstart docs.
